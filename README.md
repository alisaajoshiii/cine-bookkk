# CineBook - Movie Booking System

This is a modern movie booking web application built with React and Vite.

## Features

- **Home Page**: Featured movies hero section, "Now Showing", and "Coming Soon" lists.
- **Movie Details**: View movie info, cast, rating, and trailer background.
- **Seat Booking**: Interactive seat map with visuals, price calculation, and booking confirmation.
- **Responsive Design**: Works on Desktop and Mobile.
- **Dark UI**: Premium Netflix-style aesthetics.

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open the link (usually http://localhost:5173) in your browser.

## Project Structure

- `src/components`: Reusable components (Navbar, Footer, MovieCard).
- `src/Pages`: Page components (Home, MovieDetail, Booking).
- `src/data.js`: Mock data for movies.
- `src/index.css`: Global styles and variables.
